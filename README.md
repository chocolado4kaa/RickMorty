This is a Rick and Morty application built with **TypeScript + Vite**, **React**, **TanStack Query**, **React Skeleton**, and **React Hot Toast**.  
It fetches data from [Rick and Morty API](https://rickandmortyapi.com/) and provides a fully interactive and responsive experience.

### Features

- View characters, locations, and episodes üß¨üåçüì∫  
- Search characters by name  
- Pagination for large lists  
- Skeleton loaders while data is loading using **React Skeleton**  
- Error notifications with **React Hot Toast**  
- Responsive design for mobile and desktop üì±üíª  
- Routing between pages using **React Router**  
- Fetching data from 3 APIs: Characters, Locations, Episodes  

### Technologies Used

- **TypeScript + Vite** ‚Äì frontend framework and build tool  
- **React** ‚Äì UI library  
- **React Router** ‚Äì client-side routing  
- **TanStack Query** ‚Äì data fetching and caching  
- **React Skeleton** ‚Äì loading skeletons  
- **React Hot Toast** ‚Äì toast notifications  
- **Rick and Morty API** ‚Äì external API for characters, locations, and episodes  

### How to Run

1. Clone the repository:  

   ```bash
   git clone <your-repo-url>
   ```

3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the development server:

   ```bash
   npm run dev
   ```
5. Open your browser at `http://localhost:5173`
